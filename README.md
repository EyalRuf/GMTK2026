# Nine Lives

A 2.5D sidescroller puzzle platformer about a cat and its nine lives.

Made for **GMTK Game Jam 2026** — theme: **Countdown**.

## Tech

- Unity `6000.4.1f1`
- URP, 3D project with 2.5D gameplay
- New Input System

## Repo layout

| Path | What |
|---|---|
| `Assets/` | Game scenes, scripts, prefabs, art |
| `Claude/STATE.md` | Current state of the game — what works, what's next |
| `Claude/UNITY_CLI.md` | Unity CLI / MCP tooling notes |
| `CLAUDE.md` | Working agreement for AI-assisted development |

## Running it

Open the project in Unity `6000.4.1f1` and load the main scene from `Assets/Scenes/`.
